# CommandHandler.HelpMenu
```lua
<<string> HelpMenu> CommandHandler.HelpMenu(<CommandHandler> self)
```
This generates a help menu you can use. Here is an example output:
```
:kill
:m
:h
```

### Parameters
* `CommandHandler` Handler
> The command handler

### Returns
* `string` HelpMenu
> Explained above
